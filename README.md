# Material Required
- Arduino Uno<br/>
- Bluetooth HC-06<br/>
- Jump Wires<br/>
- Breadboard<br/>
- L298N Motor Driver<br/>
- 2 X Special Kind of Battery (18650 li-ion battery 3.7V)<br/>
- Battery Holder<br/>
- 4 X Micro motors <br/>
- 4 X Grippy Wheels<br/>
